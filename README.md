# MindPiece-Dyslexia-Support-Web-Application
MindPiece is a UI/UX design project focused on creating an accessibility-friendly web application for users with dyslexia. The project emphasizes readable interfaces, accessibility-focused design systems, and user-centered experiences to support learning and reading activities.


## Features
- Accessibility-focused interface design
- Reading-friendly typography and color palette
- Progress tracking UI
- Responsive layouts and modern UI components

## Tools & Technologies
- Figma
- UI/UX Design
- Wireframing
- Prototyping
